# 🦠 COVID-19 Data Analysis Using SQL

## 📌 Project Summary
This project is a data analysis case study on global COVID-19 data using SQL Server. It focuses on analyzing infection rates, death rates, and vaccination progress across countries and continents. The goal is to derive data-driven insights and prepare structured datasets for Tableau dashboard visualization.

---

## 🎯 Objective
To analyze large-scale COVID-19 datasets using SQL and extract meaningful insights related to:
- Infection spread trends
- Mortality rates
- Population impact
- Vaccination rollout effectiveness

This project demonstrates strong skills in SQL querying, data cleaning, aggregation, and analytical thinking.

---

## 🛠️ Tools & Technologies
- SQL Server (T-SQL)
- Microsoft Excel (Data preparation for Tableau)
- Tableau (Data Visualization)
- Dataset: COVID-19 Deaths & Vaccinations
